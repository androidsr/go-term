# Go-Term - 智能SSH终端管理器

一款功能强大的跨平台SSH终端管理工具，让服务器管理变得简单高效。通过直观的图形界面，轻松管理多个远程服务器，支持终端操作、文件管理、批量脚本执行等丰富功能。

## 🌟 主要功能特点

### 📋 服务器连接管理
- **分组管理**：按项目或用途对服务器进行分组，一目了然
- **多种认证方式**：支持密码认证和SSH密钥认证
- **快速连接**：一键连接到远程服务器，状态实时显示
- **安全存储**：服务器配置信息加密保存，保护数据安全

### 💻 智能终端操作
- **全功能终端**：基于xterm.js的真实终端体验
- **多标签页**：同时打开多个终端，轻松切换
- **自动补全**：智能命令和路径补全，提升操作效率
- **响应式界面**：终端大小自适应，窗口调整灵活

### 📁 可视化文件管理
- **图形化界面**：通过SFTP协议远程管理文件
- **拖拽操作**：支持文件上传下载，操作便捷
- **目录浏览**：直观的文件夹树形结构展示
- **批量操作**：支持多文件选择和批量处理

### 🚀 脚本自动化管理
- **常用脚本库**：保存和管理常用脚本，随时调用
- **批量执行**：一键在多台服务器上执行相同脚本
- **智能语法**：支持文件传输命令和错误控制
- **执行模式**：命令模式和脚本模式灵活切换
- **终端交互**：支持在真实终端环境中执行脚本

### 🔧 命令集管理
- **快速命令**：保存常用命令组合，一键执行
- **分类整理**：按功能对命令进行分类管理
- **快速访问**：减少重复输入，提升工作效率

## 🚀 快速开始

### 环境要求
- Go 1.25+
- Node.js 16+
- npm 或 yarn

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd go-term
```

2. **安装前端依赖**
```bash
cd frontend
npm install
cd ..
```

3. **开发环境运行**
```bash
wails dev
```

4. **构建生产版本**
```bash
wails build
```

## 📖 使用指南

### 添加和管理服务器

1. **创建服务器分组**
   - 点击左侧"服务器分组"区域的"+"按钮
   - 输入分组名称，如"生产环境"、"测试环境"等
   - 点击确认创建分组

2. **添加服务器**
   - 选择对应的服务器分组
   - 点击"添加服务器"按钮
   - 填写服务器信息：
     - 服务器名称（便于识别）
     - 主机地址（IP或域名）
     - 端口号（默认22）
     - 用户名
   - 选择认证方式：
     - 密码认证：输入登录密码
     - 密钥认证：选择私钥文件路径
   - 点击确认添加服务器

3. **连接服务器**
   - 在服务器列表中找到目标服务器
   - 点击"连接"按钮建立SSH连接
   - 连接成功后状态会显示为绿色"已连接"

### 终端操作

1. **打开终端**
   - 确保服务器已连接
   - 点击服务器行的"终端"按钮
   - 新标签页中将打开终端界面
   - 可以像使用普通SSH终端一样执行命令

2. **终端功能**
   - 支持复制粘贴操作
   - 智能命令自动补全
   - 支持常用的终端快捷键
   - 可同时打开多个终端标签页

### 文件管理

1. **打开文件管理器**
   - 确保服务器已连接
   - 点击服务器行的"文件"按钮
   - 新标签页中将打开文件管理界面

2. **文件操作**
   - 浏览远程目录结构
   - 上传本地文件到服务器
   - 从服务器下载文件到本地
   - 创建、删除、重命名文件和文件夹
   - 支持文件拖拽上传

### 脚本管理

1. **创建脚本**
   - 在左侧导航栏点击"批量脚本"选项卡
   - 点击"新建脚本"按钮
   - 输入脚本名称和描述
   - 编写脚本内容

2. **脚本语法**
   - **普通命令**：`ls -la`
   - **错误时继续执行**：`invalid_command $ne` （即使命令失败也会继续执行）
   - **文件上传**：`$upload C:\本地路径\文件.txt /远程路径/`
   - **文件下载**：`$download /远程路径/文件.txt C:\本地路径\`

3. **执行脚本**
   - 选择要执行的目标服务器（可多选）
   - 选择执行模式：
     - 命令模式：逐条执行，支持终端交互
     - 脚本模式：完整执行，保持上下文
   - 点击"执行"按钮运行脚本
   - 点击"终端执行"可在真实终端环境中交互执行

### 命令集管理

1. **保存常用命令**
   - 将常用的命令组合保存为命令集
   - 按功能分类，便于管理
   - 支持一键快速执行

2. **快速执行**
   - 从命令集列表中选择需要的命令
   - 选择目标服务器
   - 一键执行，提高工作效率

## 💡 使用技巧

### 脚本执行模式选择
- **命令模式**：适用于简单的命令序列，支持在终端中实时查看输出和交互
- **脚本模式**：适用于复杂脚本，能保持执行上下文，但无法交互

### 批量操作最佳实践
- 在执行批量脚本前，建议先在单台服务器上测试
- 使用 `$ne` 标记可以避免因个别命令失败中断整个脚本执行
- 文件传输命令会自动处理SFTP连接，无需手动配置

### 安全建议
- 定期更新服务器密码和SSH密钥
- 使用SSH密钥认证比密码认证更安全
- 敏感操作前先在测试环境验证

## 🤝 常见问题

**Q: 无法连接到服务器怎么办？**
A: 请检查服务器地址、端口、用户名是否正确，确认网络连接正常，验证认证信息是否有效。

**Q: 文件上传失败是什么原因？**
A: 可能的原因包括：本地文件路径错误、远程目录权限不足、磁盘空间不足、网络连接中断等。

**Q: 脚本执行异常如何调试？**
A: 可以查看详细的错误信息，建议先用简单的命令测试连接，然后逐步排查脚本中的问题。

**Q: 终端显示不正常怎么办？**
A: 尝试调整终端窗口大小，检查远程服务器的环境变量设置，确认终端类型兼容性。
